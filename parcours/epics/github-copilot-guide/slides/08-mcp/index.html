MCP — Model Context Protocol

Qu'est-ce que MCP ?

MCP (Model Context Protocol) est un protocole open-source créé par Anthropic pour connecter les applications d'IA à des systèmes externes de manière standardisée.

MCP est à l'IA ce que l'USB-C est aux périphériques : un connecteur universel standardisé.

Objectif

Permettre aux assistants IA d'accéder de manière standardisée à :

  • Sources de données — fichiers locaux, bases de données, APIs
  • Outils — moteurs de recherche, calculateurs, services externes
  • Workflows — prompts spécialisés, processus métier

Cas d'usage

Chatbot d'entreprise

Connectez Copilot à vos bases de données internes pour répondre à des questions métier via conversation naturelle.

Assistant de développement enrichi

Accès au filesystem, opérations Git automatisées, recherche dans la documentation, interaction avec vos APIs internes.

Analyse de données

Connexion directe à PostgreSQL ou MongoDB. L'IA génère les requêtes, analyse les résultats, propose des insights.

Workflow automatisé

Création de tickets Jira depuis une conversation, mise à jour de Notion, envoi de messages Slack — le tout orchestré par l'IA.

Configuration

Les serveurs MCP se configurent généralement via un fichier JSON :

{
  "mcpServers": {
    "filesystem": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-filesystem", "/path/to/project"]
    },
    "postgres": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-postgres"],
      "env": {
        "DATABASE_URL": "postgresql://user:pass@localhost/db"
      }
    }
  }
}
