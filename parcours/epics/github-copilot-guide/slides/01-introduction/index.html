Quick Start — Copilot en 5 minutes

1 Créer le fichier projet (2 min)

Créez .github/copilot-instructions.md à la racine de votre repo :

# [Nom du projet] — Instructions Copilot

## Stack technique
- C# 12 / .NET 8 avec ASP.NET Core
- PostgreSQL avec Entity Framework Core
- React 18 avec TypeScript

## Conventions
- Nommage : PascalCase (C#), camelCase (JS/TS)
- Commits : format Conventional Commits
- Tests : xUnit (C#), Vitest (React)

## Structure
- src/Api/ : Controllers et endpoints
- src/Domain/ : Logique métier
- src/Web/ : Frontend React
- tests/ : Tests unitaires et intégration

2 Activer dans votre IDE (1 min)

VS Code

  • Extension "GitHub Copilot" installée
  • Extension "GitHub Copilot Chat" installée
  • Ctrl+Shift+P → "GitHub Copilot: Sign In"

JetBrains (Rider, PhpStorm...)

  • Plugin "GitHub Copilot" installé (Settings → Plugins)
  • Settings → Languages & Frameworks → GitHub Copilot → Login

Visual Studio

  • Extension installée (incluse par défaut v17.10+)
  • Tools → Options → GitHub → Copilot → Sign In

3 Premier prompt efficace (2 min)

Ouvrez Copilot Chat (Ctrl+Alt+I VS Code / Alt+C JetBrains) et testez :

Crée un endpoint API qui retourne la liste des utilisateurs actifs.
- Controller ASP.NET Core
- Route GET /api/users/active
- Filtrer par IsActive == true
- Retourner un DTO avec Id, Email, FullName

Commandes essentielles à retenir

Commande Usage
/explainComprendre du code existant
/fixCorriger une erreur
/testsGénérer des tests unitaires
/docAjouter de la documentation

Raccourci magique : Sélectionnez du code + Ctrl+I (VS Code) pour un prompt contextuel.

Introduction et concepts clés

Qu'est-ce que GitHub Copilot ?

GitHub Copilot est une suite d'outils d'assistance au développement basée sur l'IA, proposée par GitHub. Elle comprend plusieurs produits :

Produit Description
Copilot dans l'IDEAssistant intégré (VS Code, JetBrains, Visual Studio)
Copilot CLIAssistant en ligne de commande
Copilot sur github.comChat web, revue de PR automatique
Copilot APIPour intégrations custom
Copilot ExtensionsConnexion à des outils tiers (MCP)
Ce guide se concentre sur l'usage dans les IDE, qui représente le cœur de l'expérience développeur au quotidien. Une section couvre également Copilot CLI pour les usages terminal.

Modes d'interaction dans l'IDE

Mode Description Cas d'usage
Complétion inlineSuggestions automatiques pendant la frappeCode répétitif, patterns connus
Copilot ChatInterface conversationnelleQuestions, refactoring, explications
Mode AgentExécution autonome multi-fichiersTâches complexes, génération de code
