GitHub Copilot CLI PREVIEW

GitHub Copilot CLI est un agent IA terminal qui apporte les capacités de Copilot coding agent directement dans votre ligne de commande. Il peut éditer du code, exécuter des commandes et interagir avec GitHub.

Installation

# macOS / Linux (Homebrew)
brew install copilot-cli

# Windows (WinGet)
winget install GitHub.Copilot

# Via npm (toutes plateformes, Node.js v22+ requis)
npm install -g @github/copilot

# Script rapide (macOS/Linux)
curl -fsSL https://gh.io/copilot-install | bash

Puis lancez copilot et authentifiez-vous avec /login.

Agents personnalisés

Copilot CLI reconnaît les agents définis dans :

Scope Emplacement
Utilisateur~/.copilot/agents/
Repository.github/agents/
Organisation.github-private/agents/

Lancement : copilot --agent=refactor-agent ou via slash command.

Delegate — Déléguer à GitHub

La commande /delegate pousse votre session vers Copilot coding agent sur GitHub :

  • Crée une draft PR automatiquement
  • Copilot continue le travail en arrière-plan
  • Vous êtes notifié pour review quand c'est prêt

OpenCode OPEN SOURCE

OpenCode est une alternative open-source à Copilot CLI. Il permet d'utiliser n'importe quel modèle IA et ne stocke aucune donnée.

Caractéristiques

Aspect Détail
LicenceOpen source
Modèles75+ providers (Claude, GPT, Gemini, modèles locaux)
ConfidentialitéNe stocke aucune donnée de code
SessionsMulti-sessions parallèles, sessions partageables
LSPIntégration Language Server Protocol automatique

Installation

# Script rapide
curl -fsSL https://opencode.ai/install | bash

# npm
npm i -g opencode-ai@latest

# Homebrew (macOS/Linux)
brew install opencode

# Windows
choco install opencode

Intégration GitHub

# Dans un commentaire d'issue GitHub :
/opencode explain this issue
/opencode fix this
